Procurando um substituto para o "Save Image as Type"?

A popular extensão "Save Image as Type" foi removida da Chrome Web Store após a descoberta de malware e comportamento indesejado. Se você dependia dela para salvar imagens no formato que precisava, você não está sozinho — milhares de usuários estão buscando agora uma alternativa confiável.

O Safe Image Converter faz exatamente o que aquela extensão fazia, com uma diferença fundamental: foi construído desde o início tendo privacidade e transparência como base. Cada linha de código é focada exclusivamente em converter imagens localmente — nada além disso.

Safe Image Converter é um substituto limpo e independente, construído do zero. Sem rastreamento, sem assinatura, sem pegadinha.

Como funciona

🖱️

Clique com o botão direito em qualquer imagem

Em qualquer página da web, clique com o botão direito em uma imagem para abrir o menu de contexto. A extensão adiciona três novas opções automaticamente.

🎨

Escolha o seu formato

Selecione Salvar como PNG, JPG ou WebP. A conversão acontece instantaneamente no seu navegador usando a Canvas API.

⬇️

Baixe na hora

Seu navegador baixa o arquivo convertido imediatamente — com um nome inteligente baseado no nome original da imagem.

Privacidade por design

A conversão de imagens acontece inteiramente dentro do seu navegador usando a Canvas API padrão. A extensão baixa a imagem fonte para desenhá-la em um canvas. Essa é a única requisição de rede que ela faz. Sem SDK de analytics, sem telemetria, sem sistema de contas e sem servidor externo para enviar dados — porque ela simplesmente não precisa de um.
