Looking for a "Save Image as Type" replacement?

The popular "Save Image as Type" extension was removed from the Chrome Web Store after reports of malware and unwanted behavior. If you relied on it to save images in the format you actually wanted, you're not alone — thousands of users are searching for a trustworthy alternative right now.

Safe Image Converter covers that core workflow and now adds direct background cleanup actions too. It was built with privacy and transparency as the foundation, and every line of code is focused solely on converting images locally — nothing else.

Safe Image Converter is a clean, independent replacement built from scratch. No tracking, no subscription, no catch.

How it works

🖱️

On any webpage, right-click an image to open the context menu. The extension adds save and background actions automatically.

🎨

Pick your format

Choose Save as PNG, JPG, or WebP, remove a white/checker background, or add a white/black background. The conversion happens instantly in your browser using the Canvas API.

Download instantly

Your browser downloads the converted file right away — named smartly from the original image filename.
